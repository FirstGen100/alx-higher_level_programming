List test module
=================
Import the class.
::
	>>> MyList = __import__('1-my_list').MyList

Checking testcases.
::
	>>> my_list = MyList()

	>>> my_list.append(5)
	
	>>> my_list.append(4)

	>>> my_list.append(3)

	>>> my_list.append(2)

	>>> my_list.append(1)

	>>> print(my_list)
	[5, 4, 3, 2, 1]

	>>> my_list.print_sorted()
	[1, 2, 3, 4, 5]

Checking empty object.
::
	>>> obj = MyList()
	>>> print(obj)
	[]

	>>> obj.print_sorted()
	[]

Checking negative values.
::
	>>> obj = MyList()
	>>> obj.append(-23)
	>>> obj.append(-1)
	>>> print(obj)
	[-23, -1]

	>>> obj.print_sorted()
	[-23, -1]
